GRIDNOTES - START HERE
======================

You do not need coding experience.

IMPORTANT - TWO DIFFERENT FILES
-------------------------------
  Install GridNotes.bat  = INSTALLER (run once to set up GridNotes)
  Open GridNotes.bat     = opens the app AFTER you installed

The download folder is NOT where GridNotes runs day to day.
After install, use the Desktop icon "GridNotes" or Open GridNotes.bat.

DOWNLOADED FROM GITHUB (ZIP / source code)?
  You will NOT find GridNotes-Setup.exe in that folder. That is normal.

  1. Extract the ZIP anywhere (Downloads, Desktop, etc.)
  2. Install Python from https://www.python.org/downloads/
     (Turn ON "Add python.exe to PATH" on the first screen.)
  3. Double-click: Install GridNotes.bat (no admin needed for default install)
  4. In the window, click "Install GridNotes" and wait until it finishes
  5. Open the app from the "GridNotes" icon on your Desktop
     (or double-click Open GridNotes.bat in this folder)

  Default install location: your user folder (no admin)
  (or D:\GridNotes if you chose D: in the wizard)

HAVE GridNotes-Setup.exe (from a Release download)?
  1. Double-click GridNotes-Setup.exe
  2. Install, then open GridNotes from the Start menu or Desktop

Full guide: open INSTALL.md in this folder.
Releases: https://github.com/troyerl/GridNotes/releases
