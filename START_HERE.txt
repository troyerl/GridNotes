GRIDNOTES - START HERE
======================

You do not need coding experience.

DOWNLOADED FROM GITHUB (ZIP / source code)?
  You will NOT find GridNotes-Setup.exe in that folder.
  That is normal. Use Install GridNotes.bat instead:
    1. Extract the ZIP anywhere (Downloads, Desktop, etc.)
    2. Install Python from https://www.python.org/downloads/
       (Turn ON "Add python.exe to PATH" on the first screen.)
    3. Double-click: Install GridNotes.bat (click Yes if Windows asks)
    4. Click "Install GridNotes", then use the Desktop icon later
  Installs to: C:\Program Files\GridNotes (default)

HAVE GridNotes-Setup.exe (from a Release download)?
  1. Double-click GridNotes-Setup.exe
  2. Install, then open GridNotes from the Start menu

Full guide: open INSTALL.md in this folder.
Releases: https://github.com/troyerl/race_book/releases
