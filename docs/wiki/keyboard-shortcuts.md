# Keyboard shortcuts

**See also:** [Wiki home](README.md) · [Drivers tab](drivers-tab.md) · [Live Mode](live-mode.md)

GridNotes supports a small set of keyboard shortcuts for common scouting actions.

---

## Global shortcuts

| Shortcut | Action |
|----------|--------|
| **Ctrl+F** | Focus **Search drivers** on the [Drivers tab](drivers-tab.md) |
| **Ctrl+L** | Toggle **Live Mode** on/off |
| **Ctrl+S** / **Cmd+S** | **Save notes** for the selected driver |

---

## Driver table

| Key | Action |
|-----|--------|
| **↑** / **↓** | Move selection between rows |
| **Enter** | Focus the scouting notes editor for the selected driver |

---

## Live Mode and Grid Walk

| Key | Action |
|-----|--------|
| **Enter** / **Space** | Expand or collapse the focused Live Mode card or Grid Walk row (details stay on screen) |

Works when a Live Mode card or Grid Walk row has keyboard focus.

---

## Marks and preferences

Use the **Liked**, **Didn't like**, and **Clear** buttons in the driver detail panel — there are no dedicated keyboard shortcuts for marks.
